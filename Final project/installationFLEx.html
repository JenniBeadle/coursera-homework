How to Create a New FieldWorks Project

  1. Start FieldWorks.
    • The Language Explorer dialog is displayed.
  1. Select Create a new project.
  2. Follow the prompts in the New Project Wizard:
  1. Project Name: Enter a name for your project.
  2. Vernacular Language Information: Enter the language name, ISO code (if you have one), and other details.
  3. Analysis: Enter the language name ...
  4. Customize Writing Systems: Set up the main writing system(s) for your language. You can add more later if needed.
  5. Anthropology (this is optional but can't be added later)
  6. Project Location: Choose where to save your project files.
  7. Finish: Click Finish to create your project.

After the wizard completes, your new project will be ready, and you can start adding lexical entries, texts, and other data.
